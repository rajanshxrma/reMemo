# reMemo

A free, simple, classic, completely ad-free memo app — say "remember that..." and it saves. Built as a teenager learning to ship real software for the first time, using Thunkable (a block-based, visual app builder — the same category as Scratch, but it compiles to real installable Android apps instead of just running in a browser sandbox).

Shipped under the developer handle **Walkin**.

## What it does

- Speech-to-text memo creation — talk, hit save, done
- Search and pin-to-top for the memos you actually use
- English by default, with Korean and Japanese support in beta
- Automatic dark theme, sunset to sunrise
- Free. No ads. No in-app purchases. Ever.

## The real timeline

No source repo ever existed for this — Thunkable is a cloud-hosted visual block editor, not something that produces a git-trackable codebase. This README (and its dates) is the permanent record instead, reconstructed from the actual emails and receipts from building it:

- **March–April 2020**: Learning Thunkable from tutorials, working through the basics of list views and data storage
- **May 30, 2020**: First version shipped to the Amazon Appstore, originally listed as "reMemo: Notes"
- **June 2020**: Added a Firebase backend
- **August 31, 2020**: Registered as a Google Play developer, brought reMemo to Play and Samsung Galaxy Store
- **September 2020**: Five rapid point releases (1.5.1 → 1.5.5) as real early users sent feedback and things got fixed fast

It stayed live and quietly supported for years after — the last user bug report on record came in **June 2023**, three years after the first release, still running on an app a teenager built with a visual block editor and no formal training.

## Why this repo exists

There's no code to check into git — this documents something that actually happened and actually shipped to three app stores, at a time before this account existed to properly record it. Consider this the historical marker.
